# glue-examples
Glue Frameowrk 5.1 예제입니다. 

## 사전 준비

local repository에 3rd-party(glue) 라이브러리를 등록하세요. 

1. GlueSDK-5.1.1-RELEASE.zip 을 download 한다.  
   https://www.solutionpot.co.kr 을 통해 배포됩니다.  

```bash
$ unzip GlueSDK-5.1.1-RELEASE.zip       # 압축풀기
Archive:  GlueSDK-5.1.1-RELEASE.zip
   creating: GlueSDK/config/
   creating: GlueSDK/lib/
   creating: GlueSDK/lib/gluelib/
   creating: GlueSDK/lib/gluestd/
   creating: GlueSDK/repo/              # mvn install:install-file 실행 참고 위치
   creating: GlueSDK/templateFolder/
   creating: GlueSDK/templateFolder/maven/
   # 중략
```

2. mvn install:install-file 수행한다.

```bash
$ cd GlueSDK/repo                       # repo 디렉토리로 이동
$ ls                                 # 파일 목록 확인
glue-core-5.1.1-RELEASE.jar
glue-core-5.1.1-RELEASE.pom
glue-framework-5.1.1-RELEASE.pom
glue-push-5.1.1-RELEASE.jar
glue-push-5.1.1-RELEASE.pom
glue-schema-5.1.1-RELEASE.jar
glue-schema-5.1.1-RELEASE.pom
glue-ucube-5.1.1-RELEASE.jar
glue-ucube-5.1.1-RELEASE.pom
glue-util-5.1.1-RELEASE.jar
glue-util-5.1.1-RELEASE.pom
push-client-0.0.1.jar
seadapter-3.1.26.jar
$                                       # 순서대로 install-file 실행
$ mvn install:install-file -Dfile=glue-framework-5.1.1-RELEASE.pom -DpomFile=glue-framework-5.1.1-RELEASE.pom
$ mvn install:install-file -Dfile=glue-schema-5.1.1-RELEASE.jar    -DpomFile=glue-schema-5.1.1-RELEASE.pom
$ mvn install:install-file -Dfile=glue-util-5.1.1-RELEASE.jar      -DpomFile=glue-util-5.1.1-RELEASE.pom
$ mvn install:install-file -Dfile=glue-core-5.1.1-RELEASE.jar      -DpomFile=glue-core-5.1.1-RELEASE.pom
```

## 실행

```bash
git clone https://github.com/poscoict-glueframework/glue-examples.git
cd glue-examples
``` 

```bash
mvn clean package -Pall              # 전체빌드

mvn clean package -Pquick-start      # quick-start 만 빌드     ( cd quick-start/;mvn package )
mvn clean package -Pcatering-pass    # catering-pass 만 빌드 ( cd catering-pass/;mvn package ) 
mvn clean package -Pjob-fcm          # job-fcm 만 빌드              ( cd job-fcm/;mvn package ) 
mvn clean package -Psaga             # saga-parent 만 빌드     ( cd saga-parent/;mvn package ) 
```

```bash
java -jar quick-start/target/demo.jar                    # quick-start 실행

java -jar catering-pass/target/catering-pass.jar         # catering-pass 실행

java -jar job-fcm/target/job-fcm.jar                     # job-fcm 실행
```
